# Self Driving Car Simulation (Intelligent System)

This project demonstrates a simple self-driving car using Arduino, ultrasonic sensor, and L293D motor driver.

## Components
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- L293D Motor Driver
- DC Motors
